All memory files live in the project's `memory/` folder so the user can see and edit them directly.

**Why:** Full visibility and control over what the AI remembers. No hidden state. If you can't see it, you can't fix it.

**How to apply:** Always save memory both to the system folder (for auto-loading) and to the project's `memory/` folder (for user access).
